# ブラックジャック

ブラウザ上で遊べるシンプルなブラックジャックゲームです。HTML、CSS、JavaScript だけで動作し、カード画像を使用した見た目のあるインターフェースを提供します。

## 機能

- 新しいゲームの開始
- ヒット / スタンド操作
- プレイヤー・ディーラーの得点表示
- 勝敗と引き分けの記録
- カード画像を使った表示

## 使い方

1. このフォルダを開きます。
2. `index.html` をブラウザで開きます。
3. 「新しいゲーム」ボタンからプレイを開始します。

## ファイル構成

- `index.html` : ゲーム画面のHTML
- `style.css` : スタイル定義
- `game.js` : ゲームロジック
- `cards/` : カード画像

## 備考

このプロジェクトは学習用のシンプルな JavaScript ゲームとして作成されています。
